The Dr. Zoe Solution

Dr. Zoe uses Artificial Intelligence and IoT-based sensors to monitor vital health parameters, analyze symptoms, and connect patients with real doctors online when needed.

It operates on a custom-built Zoe OS, developed using cutting-edge technologies. The system is user-friendly, with accessibility features like font resizing and color adjustments.

Powered by Zoe OS

Custom operating system built with HTML, CSS, JavaScript, APIs, JSON, Bootstrap, ESP8266, and Arduino Uno

Features

Comprehensive Healthcare Solutions

Advanced AI-powered features designed for complete health monitoring

Vital Monitoring

Real-time monitoring using MAX30100 sensor for pulse rate, oxygen saturation (SpO2), and BPM tracking.

Temperature Monitoring

Accurate body temperature measurement using DS18B20 sensor for fever detection and monitoring.

Weight & Height

Precise measurements using load cells for weight and ultrasonic sensors for height tracking.

Eye Test

Comprehensive vision clarity check through letter-size recognition and visual acuity assessment.

Hearing Test

Advanced hearing assessment by playing sounds of different frequencies to evaluate hearing ability.

First Aid Tutor

Learn essential first aid techniques through simple video and document-based tutorials.

AI Diagnosis

Intelligent symptom analysis and health assessment using advanced AI algorithms.

Smart Prescriptions

Automated prescription generation and printing based on comprehensive health analysis.

Doctor Connect

Seamless online connection with real doctors when needed for expert consultation.
